C# 中的委托（Delegate）类似于 C 或 C++ 中函数的指针。委托是一个类，定义了方法的类型，从而可以将方法当作另一个方法的参数来进行传递，这种将方法动态地赋给参数地做法，可以避免在程序中大量使用if-else语句。

委托（Delegate） 是 存有对某个方法的引用 的 一种引用类型变量。引用可在运行时被改变。

委托（Delegate）特别用于实现事件和回调方法。

所有的委托（Delegate）都派生自 System.Delegate 类。

声明委托（Delegate）

委托声明决定了可由该委托引用的方法。委托可指向一个与其具有相同标签的方法。声明一个delegate类型，它必须与要传递的方法具有相同的参数和返回值类型。

Socket

Posted on 2022-03-27 Edited on 2022-09-08

Socket简介

网络上两个程序通过一个双向的通信连接实现数据交换，这个连接的一端称为一个Socket。

一个Socket包含：
协议，本地IP，本地端口，远程IP，远程端口

Socket通信分为：连接方和监听方
连接方：不同的客户端使用的是不同的端口
监听方：服务器，只使用一个端口监听

Socket通信的流程

1、
服务器：创建一个Socket对象（使用API Socket），绑定本地使用的端口（使用API Bind）。
客户端：创建一个Socket对象（使用API Socket）。

2、
服务器：开启监听（使用API Listen），等等客户端接入

3、
客户端：连接服务器（使用API Connect），该过程中客户端会由系统自动分配端口

4、
服务器：接受连接（使用API Accept），成功建立连接。

5、
服务器：Send和Receive
客户端：Send和Receive
OS：自动完成数据的确认、重传等，确保传输的数据准确

6、
某一方断开连接（使用API Close）
OS：自动执行”四次握手“，关闭连接。
