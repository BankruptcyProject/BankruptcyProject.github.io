lua

lua菜鸟教程:https://www.runoob.com/lua/lua-coroutine.html

目前项目使用Lua 5.1

在默认情况下，变量总是认为是全局的

如果想删除一个全局变量或者table，只需要将变量赋值为nil

nil表示一个空值

nil 作比较时应该加上双引号 " type(X)=="nil"

boolean 类型只有两个可选值：true 和 false Lua 把 false 和 nil 看作是 false，其他的都为 true，数字 0 也是 true

使用 # 来计算字符串的长度，放在字符串前面 print(#"www.runoob.com")

使用 .. 连接两个字符串 print("Hello".."World")

Lua 里表的默认初始索引一般以 1 开始

lua 不存在continue语句

Lua 模块与包

加载模块会直接执行模块中的代码，并返回模块的返回值

Socket_Echo网络程序

Posted on 2022-03-27 Edited on 2022-11-14

客户端

创建Socket对象

Socket(AddressFamily.InterNetwork, SocketType.System, ProtocolType.Tcp)

AddressFamily：
地址族，有两种。InterNetwork，指Ipv4；InterNetworkV6，只Ipv6

SocketType:
套接字类型，有多种类型，游戏开发中最常用的是字节流套接字，Stream。

ProtocolType:
指明协议，这里使用的是TCP协议

连接Connect

客户端通过 socket.Connect(远程IP地址, 远程端口); 连接服务器。 Connect是一个阻塞方法，程序会卡住直到服务端回应（接收、拒绝或超时）。

发送消息Send

socket.Send，是阻塞方法。
该方法的参数为一个byte[]作为发送内容
该方法的返回值指明发送数据的长度

接收消息Receive

socket.Receive，是阻塞方法。
该方法的参数为一个byte[]存储接收数据
该方法的返回值指明接收数据的长度

关闭连接Close

socket.Close

服务器

绑定Bind

listenfd.Bind(ipEp)，给套接字listenfd绑定ip和端口。这里使用的是本地地址和8888号端口。

监听Listen

服务端通过listenfd.Listen(backlog)开启监听。（启动服务器）。
参数 backlog 指定队列中最多可容纳等待接受的连接数，0表示不限制。

应答Accept

开启监听后，用listenfd.Accept接受某个客户端的连接（Connect）。
返回一个Socket对象代表接收到的客户端（因为可能会接收多个客户端）。
