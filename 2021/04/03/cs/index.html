第一课

一个 C# 程序一般结构：

类一般包含多个方法。
方法定义了类的行为。
Main 方法，是所有 C# 程序的 入口点。Main 方法说明当执行时 类将做什么动作。
Console.ReadKey(); 是针对 VS.NET 用户的。这使得程序会等待一个按键的动作，防止程序从 Visual Studio .NET 启动时屏幕会快速运行并关闭。
WriteLine 是一个定义在 System 命名空间中的 Console 类的一个方法。

注：
C# 是大小写敏感的。
所有的语句和表达式必须以分号（;）结尾。
程序的执行从 Main 方法开始。

数据类型

值类型

对于一个变量有专属的一个内存地址

引用类型

对于莫个地址有不同的变量指向它
比如定义引用类型 a=10，这时候在内存中保存的是 10，当把 a 赋给 b 的时候，他们两个应用的是同一个内存空间，b 的值会保存为 a 的值，当把 b 改为 20 时，应为 a 和 b 引用的是同一个所以 a 也变成 20 了

交错数组

交错数组是数组的数组，交错数组本质上是一维数组。

参数数组

不能确定要传递给函数作为参数的参数数目

带 params 关键字的参数类型必须是一维数组，不能使用在多维数组上
带 params 关键字的参数必须是最后一个参数，并且在方法声明中只允许一个 params 关键字
我们通常的入口函数 static void Main(string [] args) 就是用数组来作形参的，它实际可以写成 static void Main()，不需要参数的传入

结构体

定义在类外？

结构体中声明的字段无法赋予初值

类的对象是存储在堆空间中，结构存储在栈中。堆空间大，但访问速度较慢，栈空间小，访问速度相对更快。

结构可带有方法、字段、索引、属性、运算符方法和事件，但结构成员不能指定为 abstract、virtual 或 protected

析构函数的名称是在类的名称前加上一个波浪形（~）作为前缀，它不返回值，也不带任何参数。

类的静态成员或方法

我们可以使用 static 关键字把类成员定义为静态的。当我们声明一个类成员为静态时，意味着无论有多少个类的对象被创建，只会有一个该静态成员的副本。

关键字 static 意味着类中只有一个该成员的实例。静态变量用于定义常量，因为它们的值可以通过直接调用类而不需要创建类的实例来获取。静态变量可在成员函数或类的定义外部进行初始化。你也可以在类的定义内部初始化静态变量。

你也可以把一个成员函数声明为 static。这样的函数只能访问静态变量。静态函数在对象被创建之前就已经存在。
