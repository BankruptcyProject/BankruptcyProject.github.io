一般项目结构

Posted on 2024-06-11

以空项目empty举例

empty
res 代码以外的其它资源（包含图片、音频等等）
src 脚本文件
debug.log 在quick-x-player上调试生成的控制台日志信息
UserDefault.xml 本地化存储文件
README 开发人员及版本等信息文件
src
app 当前项目的各个模块、场景，例如gameplay
com 是我们项目中一些可以封装出来共用的一些组件
lang 国际化配置的文件
plugin 插件
game.lua 游戏的启动文件，也是游戏的入口
game_init.lua 游戏的初始化操作
config.lua 项目的初始化操作，如：导入框架文件之类的
main.lua 程序的唯一入口，我们的项目代码就是从这里开始执行的

app/gameplay模块 1、“scene”： 模块的场景类文件 “XxxScene.lua”:主要展示场景的内容 2、“layer”： 场景的所有的层文件 "MainLayer"：主要逻辑层，在此添加各个node，以及逻辑 "ButtonLayer"：按钮层，置于最顶层的按钮存放于此 "BackgroundLayer"：背景层，比较少用 3、“node”： 层里面所有的精灵节点的类文件

res
“android_zh.json”、“ios_en.json” Android、iOS的配置文件
“bb.bin”和“bb_64.bin” bbframework的框架的压缩版，可以在bbframework_lua_3项目里里面看到框架的源代码（源代码链接在开头）
“box.bin”和“box_64.bin” 框架的盒子，即：从游戏启动的开始到开场动画、公司Logo、巴士车等等
db 里面是数据库文件
i18n 存放国际化资源，十八国语言
img 存放图片、粒子文件等等
mov 存放视频，现在公司盒子的开场动画就放在这里
plg 放置框架的一些插件
snd 存放音频资源
fnt 存放字体文件，注：公司目前所有购买的字体只能在美术图上使用，不能在程序中使用
VERSION 版本信息

注：bbframework在读取图片资源的时候会先去国际化文件夹里面根据当前设备语言环境读取相应的资源，当找不到该资源后才会去“res/img”里面去读取，音频等其他资源也是类似

img 1、x2：通常称为“2倍图”、“普清图”，主要播放安卓平台以及ios中分辨率较低的机子，如ipad mini1 2、x4：通常称为“4倍图”或“高清图”，主要播放平台为ios分辨率较高的机子，如ipad,iphone6之后的机子 3、common：主要放置游戏中x2,x4可以共用的资源，如：粒子，tilemap的文件，骨骼的xml等等 4、xx：主要放置框架中x2,x4公共的资源 注：开发线是先制作X4再通过工具缩图至X2， 对于资源的调用框架默认自动索引到x2/x4的目录下，如：“bbframework/res/img/x4/gameplay/bg.png”，我们只需写相对路径为为：“gameplay/bg.png”

snd 1、effect：存放音效，如：打斗时武器刀剑撞击在一起的声音等 2、music：存放背景音乐 注：对于音频的调用，框架默认自动索引到snd目录下，如：“bbframework/res/snd/effect/17901001.mp3”，我们只需写相对路径为：“effect/17901001.mp3”

lua

Posted on 2024-06-11

lua菜鸟教程:https://www.runoob.com/lua/lua-coroutine.html

目前bbframework使用Lua 5.1，不支持某些更新的lua的功能

在默认情况下，变量总是认为是全局的

如果想删除一个全局变量或者table，只需要将变量赋值为nil

nil表示一个空值

nil 作比较时应该加上双引号 " type(X)=="nil"

boolean 类型只有两个可选值：true 和 false Lua 把 false 和 nil 看作是 false，其他的都为 true，数字 0 也是 true

使用 # 来计算字符串的长度，放在字符串前面 print(#"www.runoob.com")

使用 .. 连接两个字符串 print("Hello".."World")

Lua 里表的默认初始索引一般以 1 开始

lua 不存在continue语句

Lua 模块与包

加载模块会直接执行模块中的代码，并返回模块的返回值

Posted on 2024-04-22

System.Net.Http

提供一个类，用于从 URI 标识的资源发送 HTTP 请求和接收 HTTP 响应。

HttpClient client = new HttpClient();

属性

BaseAddress 获取或设置发送请求时使用的 Internet 资源的统一资源标识符 (URI) 的基址。
DefaultProxy 获取或设置全局 HTTP 代理。
DefaultRequestHeaders 获取与每个请求一起发送的标题。
DefaultRequestVersion 获取或设置对此 HttpClient 实例发出的后续请求使用的默认 HTTP 版本。
DefaultVersionPolicy 获取或设置便捷方法中隐式创建的请求的默认版本策略，例如 GetAsync(String)和 PostAsync(String, HttpContent)。
MaxResponseContentBufferSize 获取或设置读取响应内容时要缓冲的最大字节数。
Timeout 获取或设置请求超时前等待的时间跨度。

方法

展开表

Dispose() 释放由 HttpMessageInvoker使用的非托管资源和托管资源。(继承自 HttpMessageInvoker)
GetAsync(String) 以异步操作将 GET 请求发送给指定 URI。
GetAsync(String, CancellationToken) 用以异步操作的取消标记发送 GET 请求到指定的 URI。
GetAsync(String, HttpCompletionOption) 用以异步操作的 HTTP 完成选项发送 GET 请求到指定的 URI。
GetAsync(String, HttpCompletionOption, CancellationToken) 用以异步操作的 HTTP 完成选项和取消标记发送 GET 请求到指定的 URI。
PostAsync(String, HttpContent) 以异步操作将 POST 请求发送给指定 URI。
PostAsync(String, HttpContent, CancellationToken) 用以异步操作的取消标记发送 POST 请求。
PostAsync(Uri, HttpContent) 以异步操作将 POST 请求发送给指定 URI。
PostAsync(Uri, HttpContent, CancellationToken) 用以异步操作的取消标记发送 POST 请求。
PutAsync(String, HttpContent) 以异步操作将 PUT 请求发送给指定 URI。
PutAsync(Uri, HttpContent, CancellationToken) 用以异步操作的取消标记发送 PUT 请求。
Send(HttpRequestMessage) 发送带有指定请求的 HTTP 请求。
Send(HttpRequestMessage, CancellationToken) 发送带有指定请求和取消标记的 HTTP 请求。
SendAsync(HttpRequestMessage) 以异步操作发送 HTTP 请求。
SendAsync(HttpRequestMessage, CancellationToken) 以异步操作发送 HTTP 请求。

Spine动画显示错乱问题

图片资源过多导致spine导出需要用到多个图集的时候，如果一个动画播放的图片在两个图集，就可能出现显示错乱的问题。

原因是一个SkeletonGraphic默认只有一个Canvasenders。

可以先尝试在打包设置中修改最大宽高，勾选去除X,Y轴空白区域，尽可能让资源在一张图集中如果是由于多套皮肤导致的，可以根据目录，按照文件夹分组打包。

如果不能分成多文件夹可以勾选Advanced->Multiple CanvasRenders，这样单个SkeletonGraphic也可以有多个CanvasRender进行渲染

unity AnimationCurve

Posted on 2024-03-26 Edited on 2024-03-27

来自UnityEngine的类 AnimationCurve

在检查窗口中创建曲线

在脚本中定义一个 AnimationCurve 类型的变量（public），挂载这个脚本，可在检查窗口中手动调整曲线

曲线是普通的贝塞尔曲线，控制方式与Blender中相差不大

每一个控制点有的参数

使用代码创建曲线

AnimationCurve可以理解为2部分。(1)控制点序列 (2)曲线的循环模式

一：控制点序列

创建控制点序列：Keyframe[] ks = new Keyframe[3];

曲线中加入键序列：AnimationCurve curve = new AnimationCurve(ks);

获取曲线中的键序列：curve[index] 或者 curve.keys[index]

添加单键：curve.Addkey(time,value)

删除单键：curve.RemoveKey(index)

二：曲线的循环模式

anim.preWrapMode = WrapMode.Loop;

anim.postWrapMode = WrapMode.Once;

三：控制点

Keyframe kf = new Keyframe(time,value);

kf.inTangent = 45;

kf.outTangent = 45;

四：一些函数 public float Evaluate(float time)
获取当前time位置对应曲线上的value

public int AddKey(Keyframe key)
public int AddKey(float time, float value);
添加一个关键帧

public int MoveKey(int index, Keyframe key)
移动一个关键帧到指定index处

public void RemoveKey(int index);
移除一个关键帧

public void SmoothTangents(int index, float weight);
平滑化指定索引的关键帧的in和out tangents,weight：The smoothing weight to apply to the keyframe’s tangents.
