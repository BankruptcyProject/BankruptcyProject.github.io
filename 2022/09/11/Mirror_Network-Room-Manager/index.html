Mirror_Network Room Manager

Posted on 2022-09-11 Edited on 2022-11-14

Network Room Manager是Network Manager的派生类，拥有Network Manager的所有功能，用于做游戏大厅。

属性

Show Room GUI
显示房间的默认OnGUI控件。

Min Players
开始游戏所需的最小玩家数。

Room Player Prefab
玩家进入房间时要创建的预制体（需要预制体需要有 Network Room Player component组件）。

Room Scene
The scene to use for the room.

Gameplay Scene
The scene to use for main game play.

pendingPlayers
包含准备开始游戏的玩家的列表。

roomSlots
管理房间中已连接客户端插槽（slot）的列表。

allPlayersReady
Bool 变量，表示所有玩家是否准备好开始比赛。
当玩家调用“CmdChangeReadyState”时，该值会发生变化，表示为true或false，当新客户端连接时，将设置为false。
