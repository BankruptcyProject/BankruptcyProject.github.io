Network Identity

Network Identity 组件是 Unity 网络高级 API 的核心。它控制游戏对象在网络上的唯一身份，并使用该身份使网络系统知道该游戏对象。

Server Only
勾选此复选框以确保 Unity 仅在服务器上生成游戏对象，而不是在客户端上。

Mirror 不支持嵌套游戏对象上的网络身份。即父对象和子对象中只能有一个Network Identity。

实例化的网络游戏对象

使用 Mirror 的服务器权限网络系统，服务器必须使用 NetworkServer.Spawn 生成具有网络身份的网络游戏对象。这会自动在连接到服务器的客户端上创建它们，并为它们分配一个 NetId。

必须在运行时生成的任何 Prefab 上放置一个 Network Identity 组件，以便网络系统使用它们。
