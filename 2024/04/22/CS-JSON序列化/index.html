CS_JSON序列化

Posted on 2024-04-22

System.Text.Json命名空间提供用于序列化和反序列化 JavaScript 对象表示法 (JSON) 的功能。

序列化是将对象状态（即其属性的值）转换为可存储或传输的形式的过程。 序列化形式不包含有关对象的关联方法的任何信息。 反序列化从序列化形式重新构造对象。

https://learn.microsoft.com/zh-cn/dotnet/standard/serialization/
